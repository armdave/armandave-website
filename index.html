Exhaustive Trust Algorithm (ETA): Mitigating the Flood & Loot Attack of the Lightning Network

Lightning Network is a Layer 2 solution on top of the Bitcoin blockchain that that aims to facilitate exponentially faster transactions without sacrificing the trustless security of the blockchain. A security flaw of the Lightning Network known as the Flood & Loot Attack was discovered in 2020. We designed the Exhaustive Trust Algorithm (ETA) and proved that an implementation of ETA would make Floot & Loot unprofitable.
